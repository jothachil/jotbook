Everything you need, nothing you don't

Built for people who just want to write.

Auto-save

Notes save automatically with a 600ms debounce. Nothing is ever lost, even if you close the app mid-sentence.

Markdown-friendly

Write in plain text with a monospace editor. Toggle preview mode to see rendered markdown with syntax highlighting.

Export to .md

Export any note as a standard Markdown file via a native save dialog. Your notes, your files.

Local-first storage

All notes live in a SQLite database on your machine. No cloud sync, no accounts, no data leaving your computer.

Keyboard-first

New note with ⌘N, toggle sidebar with ⌘B. Right-click context menus for everything else.

Native dark theme

A carefully tuned dark palette with an orange accent. macOS-style title bar with native traffic light controls.
